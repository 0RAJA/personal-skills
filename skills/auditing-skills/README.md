# 安全审计技能

## 概述

`auditing-skills` 是一套用于安全审查外部代码、脚本和技能的标准化流程。在运行任何外部来源的代码之前，通过强制性的安全审计来识别并拦截潜在威胁，防止恶意逻辑运行、凭证窃取和系统破坏。

## 核心能力

- 关键词扫描与敏感路径检测
- 混淆代码还原与深度逻辑分析
- 沙箱隔离验证引导

## 适用场景

- 下载第三方技能后的安全审查
- 运行未知脚本前的风险评估
- 外部代码的安全性评估

## 安全提示

本审计流程仅作为辅助检查手段，无法保证 100% 安全。它能有效识别常见风险，但无法覆盖所有高级混淆攻击或未知漏洞（Zero-day）。请始终谨慎使用第三方 Skills，并在隔离环境中测试。
