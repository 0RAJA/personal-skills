# 深度技术主题学习

## 概述

`deep-topic-learning` 是一套面向复杂技术主题的结构化学习方法论。将「难学的技术」拆解为可重复的五段式工作流，帮助学习者从全局到细节逐步深入，同时把学习过程中的认知、纠错和踩坑系统性地沉淀为可检索的长期资产。

## 核心能力

- 五段式循环学习：建地图 → 拆模块跑通 → 源码对齐 → 主动总结纠错 → 疑惑档案
- 四层文档体系：地图层、模块层、真相层、沉淀层
- 内置学习沉淀模板（个人总结、FAQ、Troubleshooting）

## 适用场景

- 系统性学习大型代码库、新框架、多层架构（如 Agent、RAG、分布式系统）
- 需要将学习成果沉淀为 README、分章笔记、Notebook 教程或团队分享
- 学习过程中反复踩坑，需要可追溯的排错记录
